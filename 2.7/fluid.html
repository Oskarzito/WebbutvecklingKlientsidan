Position

position används för att positionera ut element. Man kan definiera dessa värden för position enligt följande:

  1. static: default-positionen för element.
  2. relative: elementet är positionerat relativt till dess normala position. Till exempel är detta list-item relativt positionerat enligt: left: 40px;
  3. fixed: elementet är positionerat relativt till dess viewport. Och kommer stanna där även om man exempelvis scrollar. Se längst ner till höger på denna sida.
  4. absolute: elementet är positionerat relativt till dess närmast positionerade förälderelement. Om inget såndant element finns placeras absolut positionerade element enligt dokumentets body. Dessa tre kolumner ligger i en container som är relativt positionerad vilket gör att texten längst ned till vänster (blå border) kan placeras ut precis där man vill ha den.
  5. sticky: elementet placeras beroende på var på sidan man är. Till exempel, när man scrollar på sidan följer sticky-element med. Se rubriken (Position) högst upp i denna kolumn som alltid befinner sig synlig när man scrollar (till skillnad från texten i kolumnen).

Tillsammans med top, left, bottom och right positionerar man ut element.

Float och clear

Om man tillämpar attributet float på ett elemet kommer andra element, med mera, runtom att flyta bredvid elementet. De tre kolumnerna på denna sida har alla attributet float: left; vilket gör att dessa hamnar på rad. Med attributet clear kan man förbjuda element att flyta bredvid varandra. Footern på denna sida har attributet clear: both; vilket gör att den hamnar helt själv längst ned på sidan utan något bredvid.

width och height

width och height är vad som avgöra bredd och höjd på HTML-element. För dessa attribut finns även specifikare regler som kan definieras: min-width, min-height, max-width samt max-height.

min-height definierar den minsta höjden ett element ska vara. Det kommer inte att bli mindre än denna höjd när man zoomar in och ut. max-height definierar den största höjden ett element kan vara. Elementet kommer inte bli större än maxhöjden. Samma regler gäller för min-width och max-width fast för bredden. För exempel på min- och maxhöjd, testa dra i textaren i högra kolumnen.
