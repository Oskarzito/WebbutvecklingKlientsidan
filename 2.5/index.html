2.1 Navigering

Länktillstånd och a:link, a:active, a:visited, a:hover

Länkar kan befinna sig i olika tillstånd. Tillstånden beror på om länken använts eller ej och så vidare. Dessa tillstånd är:

Link
- En helt vanlig, ej klickad (ej besökt), länk.
Visited
- En länk som klickats på (besökts).
Hover
- Länkens tillstånd när man håller muspekaren ovanför den.
Active
- Tillståndet precis när man klickar på länken.

Om vi vill ändra default-stilen hos länkars tillstånd kan vi i CSS använda oss av följande selektorer: a:link, a:visited, a:hover, a:active Dessa skall även definieras i denna inbördes ordning i CSS-filen!

Cursor

Cursor representerar muspekaren. Man kan välja att ändra den muspekare som visas om man till exempel håller pekaren ovanför ett HTML-element. Testa hoovra med muspekaren övanför denna text.
