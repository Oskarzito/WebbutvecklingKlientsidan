3.4.5 Formulärhantering

För att lägga mer tid på validering och formulärhantering kommer stilsättning ske med Bootstrap.

Guide:
  1. Namn kan vara vad som (ej tomt)
  2. Email måste vara minst ett tecken samt sluta på '@su.se'
  3. 'Annat' måste börja med stor bokstav samt vara minst 3 tecken
  4. Man måste klicka i att man accepterar
  5. Frivilligt att checka i 'Saknar du något?'. Gör man det gäller samma inputregler som för 'Annat'

Drinkmeny!
